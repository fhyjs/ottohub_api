# OttoHub api Client 1.1.1
## Features

- [x] Auth
- [x] Video
- [x] Blog
- [x] User
- [x] Following
- [x] Msg
- [x] thumbs-up/favourite
- [x] content management
- [x] report & audit
- [x] comment
- [x] profile
- [x] upload
- [x] system info
- [x] damaku

## Usage

Add to your build.gradle.

```
repositories {
    maven {        
        url 'https://maven.hanana2.link:2053/releases'
    }
}
...    
implementation "org.eu.hanana.reimu.lib.ottohub:ottohub_api:1.1.1"
```
